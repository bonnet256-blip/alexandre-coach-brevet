☁️ Connexion Google Sheets

Suivez ces 5 étapes pour synchroniser les sessions d'Alexandre dans le cloud.

1. Ouvre Google Sheets et crée un nouveau classeur nommé Alexandre Coach Brevet
2. Dans le classeur, clique sur Extensions → Apps Script
3. Supprime tout le code existant, colle le contenu du fichier google-apps-script.js fourni, puis sauvegarde (Ctrl+S)
4. Lance la fonction testInit() une fois (bouton ▶ en haut) pour créer les onglets automatiquement
5. Clique sur Déployer → Nouveau déploiement
Type : Application Web · Accès : Tout le monde
